# WWDC Registry

### Standard WWDC Structure
- Overview
- Video Chapters with timestamps
- Resources - links to other apple resources
- Related Videos - Usually other WWDC videos
- Transcript
- Code - code from the video in code snippets
- Summary - Summary of the transcript usually not complete. Can be ignored
