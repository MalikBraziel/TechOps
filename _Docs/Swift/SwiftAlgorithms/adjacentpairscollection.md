# AdjacentPairsCollection

A collection of adjacent pairs of elements built from an underlying collection.

```
struct AdjacentPairsCollection<Base> where Base : Collection
```

[AdjacentPairs.swift](https://github.com/apple/swift-algorithms/blob/1.2.1/Sources/Algorithms/AdjacentPairs.swift#L136 "Open source file for AdjacentPairs.swift")

## [Overview](https://swiftpackageindex.com/apple/swift-algorithms/1.2.1/documentation/algorithms/adjacentpairscollection\#overview)

Use the `adjacentPairs()` method on a collection to create an `AdjacentPairsCollection` instance.
