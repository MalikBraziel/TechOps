# AdjacentPairsSequence

A sequence of adjacent pairs of elements built from an underlying sequence.

```
struct AdjacentPairsSequence<Base> where Base : Sequence
```

[AdjacentPairs.swift](https://github.com/apple/swift-algorithms/blob/1.2.1/Sources/Algorithms/AdjacentPairs.swift#L70 "Open source file for AdjacentPairs.swift")

## [Overview](https://swiftpackageindex.com/apple/swift-algorithms/1.2.1/documentation/algorithms/adjacentpairssequence\#overview)

Use the `adjacentPairs()` method on a sequence to create an `AdjacentPairsSequence` instance.
